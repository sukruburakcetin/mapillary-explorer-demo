## About

This repository hosts the **live demonstration** of the **Mapillary Explorer Custom Widget** built for **Esri Experience Builder Web Maps**.

The widget allows users to explore and interact with **Mapillary street-level imagery** directly inside ArcGIS Experience Builder applications.

## What is Mapillary?

[Mapillary](https://www.mapillary.com/) is a platform that provides street-level imagery and geospatial data collected by users worldwide.

It enables developers and GIS professionals to:

- Access crowdsourced street imagery
- Perform visual map analysis
- Integrate imagery into mapping applications

---

## What is Esri Experience Builder?

[ArcGIS Experience Builder](https://www.esri.com/en-us/arcgis/products/arcgis-experience-builder/overview) is a web application builder that allows users to create responsive web apps using maps, widgets, and data sources.

This project extends Experience Builder by providing a custom widget for Mapillary integration.
